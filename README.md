Image-to-Ascii
==============

An extremely simple utility to convert any image to Ascii

A short example is:
```
image = ascii_converter.Ascii(10,'reddit.jpg')
```
Then, in the main event loop of your code, add the following:
```
image.create_image(surface)
```
